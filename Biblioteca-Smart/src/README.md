# Biblioteca Smart

Progetto Java da console per la gestione di una biblioteca.

## Funzionalità
- Aggiunta libri
- Ricerca per ID
- Ricerca per autore
- Modifica libro
- Prestito e restituzione
- Salvataggio su file
- Caricamento automatico da file

## Tecnologie usate
- Java
- ArrayList
- FileWriter
- FileReader
- BufferedReader
- Scanner

## Obiettivo del progetto
Allenamento pratico sulla programmazione ad oggetti e gestione file in Java.
